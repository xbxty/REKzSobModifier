# REKzSobModifier
## Introduction
游戏Katana Zero的Sob修改器，支持Sob数据的读取和修改
## Installation and Usage
仅支持windows(x64)环境下安装和使用
<br />下载REKzSobModifier.exe程序,点开即可使用

2. 仅支持'Action Only'的模式下使用
3. 本修改器不会视更改的时间和死亡数来更改获得奖牌的等级
